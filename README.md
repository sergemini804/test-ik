AI Diagnostic System: Исследовательская культура педагога

Привет! Это проект чат-бота для Telegram, который помогает учителям оценить свою готовность к исследовательской деятельности. Бот проводит комплексную диагностику по методикам «Сириуса», анализирует ответы и дает персональные рекомендации.

Главная «фишка» — использование искусственного интеллекта для оценки открытых ответов, где нет вариантов «да/нет».

Что умеет бот?

4 блока диагностики: от мотивации и ценностей до конкретных знаний методологии.

Умный анализ (AI): использует нейросеть (Gemini) для интерпретации эссе и свободных ответов.

Уровни развития: определяет, на каком вы этапе — репродуктивном, частично-поисковом или творческом.

Админка: выгрузка всех результатов в красивый Excel-файл для методистов.

Память: сохраняет все данные в локальной базе SQLite.

Как запустить проект

1. Требования

Убедитесь, что у вас установлен Python 3.9+ и менеджер пакетов pip.

2. Установка зависимостей

Сначала склонируйте репозиторий и установите нужные библиотеки:

pip install -r requirements.txt


3. Настройка окружения

Создайте в корне проекта файл .env (или переименуйте образец) и заполните его:

API_TOKEN=ваш_токен_от_BotFather
AI_API_URL=ссылка_на_api_интерфейс
AI_API_KEY=ваш_ключ_для_нейросети
AI_MODEL=название_модели
ADMIN_IDS=ваш_id_в_телеграм_через_запятую


4. Запуск

Просто запустите главный скрипт:

python project1.py
