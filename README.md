AI Diagnostic System: Исследовательская культура педагога

Серверная часть Telegram-бота для диагностики профессиональных компетенций учителей. Проект разработан для образовательной экосистемы «Сириус».

Архитектура построена на асинхронном стеке технологий (AsyncIO), поддерживает высокую нагрузку и реализует защиту данных через симметричное шифрование.

Функциональные возможности

Гибридная диагностика:

Классическое тестирование (выбор вариантов).

Анализ открытых ответов с помощью нейросети (LLM Gemini) через API. Оценивается смысловая нагрузка и уровень рефлексии.

Безопасность данных:

Реализовано сквозное шифрование (AES/Fernet) для ФИО пользователей и результатов тестов. В базе данных информация хранится в зашифрованном виде (BLOB).

Использование переменных окружения для хранения ключей и токенов.

Производительность (High Load):

Полностью асинхронный код.

Единая сессия  для HTTP-запросов к API нейросети.

Оптимизация SQLite: включен режим WAL и увеличенный кэш.

Middleware для защиты от флуда.

Администрирование:

Автоматическая генерация Excel-отчетов с расшифровкой данных «на лету».

Выполнение тяжелых задач (генерация файла) в отдельном потоке.

Технологический стек

Python 3.10+

aiogram 3.x — асинхронный фреймворк для Telegram API.

aiosqlite — асинхронная работа с базой данных.

aiohttp — клиент для запросов к AI API.

cryptography — библиотека для шифрования данных.

openpyxl — формирование отчетов.

Установка и запуск

1. Клонирование и зависимости

Установите необходимые библиотеки:

pip install aiogram aiohttp aiosqlite openpyxl python-dotenv cryptography


2. Конфигурация (.env)

Создайте файл .env в корне проекта. Для работы системы обязательно требуется сгенерировать ключ шифрования.

Получение ключа (выполнить в Python):

from cryptography.fernet import Fernet
print(Fernet.generate_key().decode())


Пример содержимого .env:

API_TOKEN=ваш_токен_телеграм
AI_API_URL=[https://generativelanguage.googleapis.com/v1beta/chat/completions](https://generativelanguage.googleapis.com/v1beta/chat/completions)
AI_API_KEY=ваш_api_key
AI_MODEL=gemini-pro
ADMIN_IDS=12345678,87654321
CIPHER_KEY=вставьте_сгенерированный_ключ_сюда


Важно: Утеря CIPHER_KEY приведет к невозможности расшифровать сохраненные в базе данные.

3. Запуск

python project1.py
